# Nutriatleta

Proyecto de sitio estático HTML.

## Flujo de despliegue

1. Sube este repositorio a GitHub.
2. Conecta el repositorio a Netlify.
3. Cada vez que hagas push a GitHub, Netlify desplegará automáticamente la web.

## Configuración Netlify

- Directoria de publicación: `.`
- Comando de construcción: ninguno
- Archivo de configuración: `netlify.toml`
